# Invoke-DNSResolve

A simple script meant to speed-up the process of resolving IP ranges to hostnames

## Usage

```
iex(new-object net.webclient).downloadstring('https://raw.githubusercontent.com/Leo4j/Invoke-DNSResolve/refs/heads/main/Invoke-DNSResolve.ps1')
```

```
Invoke-DNSResolve -IP 192.168.0.0/24
```
